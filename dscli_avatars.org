# ========== 使用说明 ==========
使用说明：
1. 选择喜欢的方案
2. 复制方案内容（不包括标题和空行）
3. 在微信中设置为头像
4. 建议使用等宽字体查看效果

推荐方案：
- 方案1：最简洁，适合小尺寸
- 方案3：有终端特色，体现dscli特点
- 方案5：代码块风格，适合开发者
- 方案7：命令行风格，直观易懂

提示：微信头像尺寸较小，建议选择简洁的设计，避免细节过多看不清。
