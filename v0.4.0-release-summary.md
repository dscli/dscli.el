## 主要特性

### 🚀 Emacs内置编辑器支持
- 通过环境变量启用Emacs内置编辑器
- 必须设置: `DS_CLI_USE_EMACS_EDITOR=1`, `INSIDE_EMACS=t`, `EMACS=1`
- 可选检查: EDITOR/VISUAL 环境变量警告

### 🔧 改进的进程管理
- 彻底解决进程二次杀不死问题
- 健壮的进程终止逻辑: 中断信号 → 等待 → 强制终止
- 更好的用户反馈和进程状态检查

### 🐛 重要修复
- 环境变量设置逻辑改进
- 输入缓冲区清理优化
- 动画模块语法错误修复
- 进程模块函数定义修复
- 格式问题修复（空行、光标位置等）

## 配置示例

```emacs-lisp
;; 启用Emacs内置编辑器
(setenv "DS_CLI_USE_EMACS_EDITOR" "1")
(setenv "INSIDE_EMACS" "t")
(setenv "EMACS" "1")

;; 动画间隔配置
(setq dscli-animation-interval 0.3)

;; 基本配置
(setq dscli-executable "dscli")
(setq dscli-chat-model "deepseek-chat")
(setq dscli-disable-color t)
(setq dscli-convert-markdown-to-org t)
```

## 使用方法

```emacs-lisp
;; 启动聊天
M-x dscli-chat

;; 中断进程
C-c C-c (在输出缓冲区)

;; 启动新会话
C-c C-n (在输出缓冲区)
```
